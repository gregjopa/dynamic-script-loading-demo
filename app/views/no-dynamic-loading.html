Demo using plain script tags

This page demonstrates how "defer" works by using plain script tags. It loads two script tags:

  • fast-script - downloads after 1 second and given lower priority by using defer
  • slow-script - downloads after 10 seconds and given higher priority

The slow-script does not use async or defer so it's given higher priority. It will block DOMContentLoaded and prevent fast-script from executing until it's done loading and executing.
